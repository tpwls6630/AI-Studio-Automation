이 문제를 풀기 위한 사고 과정을 원소 단위로 분석해 줘. 여기서 사고 과정이란, 어떤 정보들을 가공하여 문제 풀이에 사용될 수 있는 새로운 정보를 도출해내는 것을 한 단위로 볼 거야. 이 때 한 사고 과정에서 사용된 내용 지식, 문제 풀이 전략이 무엇인지도 각 단계별로 서술해.

각 사고과정에서 쓰인 정보 가공의 방식을 일반화된 명칭으로 표현해 줘. 비슷한 해결 방식을 가진 다른 문제들에 태그로 사용할 거야.

각 사고과정은 json파일로 저장할 수 있도록 json으로 정리해줘. 

각 사고과정은 다음과 같은 항목으로 이루어져야 해:
- 태그(제목) : 각 사고과정에서 쓰인 정보 가공의 방식의 일반화된 명칭. 비슷한 해결 방식을 가진 다른 문제의 사고과정에서 사용될 태그이므로 일반화되어야 함.
- 입력 정보
- 도출된 새로운 정보
- 사고 과정 (정보 가공)
- 내용 지식 : 문제 해결에 사용된 핵심 수학 개념 (레퍼런스 : 2015수학과_성취기준-51-122.pdf) (예: 수학(상) - 방정식과 부등식 - 연립방정식)
- 문제 풀이 전략 : 해당 사고 과정을 인간의 관점에서 떠올리기 위한 추론 아이디어, 해결 아이디어
